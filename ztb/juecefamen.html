不同招标类型 · 注意事项手册

拿到招标文件后，第一道分类研判——"这份标属于哪种采购方式 + 哪个行业赛道 → 雷埋在哪 → 该不该投"

定位：决策闸门层核心知识库 复用：销售S1-S5 / 产品P3 / 运营O1-O2 方法：两条轴定位法

前言为什么"分类"是投标第一课

投标最常见的失误不是"标书写得差"，而是搞错战场——把货物标的逻辑套到服务标上，把公开招标的策略用到竞争性磋商里，把政府采购的政策红线当工程标处理。

核心方法论：两条轴定位法

┌─────────────────────────────────────┐ 采购方式轴 │ 公开招标 / 邀请招标 / 竞争性磋商 │ (怎么招) │ 竞争性谈判 / 询价 / 单一来源 │ └─────────────────────────────────────┘ ✕ (交叉定位) ┌─────────────────────────────────────┐ 行业类型轴 │ 工程 / 货物 / 服务 / 政府采购 │ (招什么) │ EPC / PPP │ └─────────────────────────────────────┘
两条轴交叉点 = 死亡陷阱分布。同一张招标文件，在不同象限里雷埋在不同位置。投标助手的核心能力，就是先帮你定位象限，再告诉你这个象限专属的雷区。

第一章采购方式六大门派

1.1 公开招标

依法必招 · 主战场
适用场景
  • 限额以上货物/工程/服务采购
  • 招投标法强制公开项目
  • 国企、事业单位大宗采购
核心机制
  • 公开发布招标公告
  • 综合评分法（商务+技术+价格）
  • 最低价 ≠ 中标
死亡陷阱
  • 实质性条款漏响应即废标（第一大废标原因）
  • 投标保证金截止前未到账（到账≠汇出）
  • 投标文件递交超时（差一分钟即废）
  • 资格后审不过（证书/业绩/社保）
  • 围标串标认定（IP/MAC/报价规律）
报价策略
价格分反推限价；商务技术分先算赢面；梯度报价不报最低
决策清单
  • 资质门槛齐备？
  • 类似业绩达标？
  • 项目经理可锁定？
  • 限价有利润空间？
  • 实质性条款能逐条响应？

1.2 邀请招标

限制性竞争
适用场景
  • 仅少数投标人具备能力
  • 保密/安全/专利限制
  • 依法不公开的特殊项目
核心机制
  • 向3家以上发邀请书
  • 不公开发布公告
  • 评标同公开招标
死亡陷阱
  • 邀请程序违法被质疑（名单不透明）
  • 围标串标嫌疑放大（关联方聚集）
  • 资格预审材料造假→ 黑名单
  • 受邀方不足3家流标
报价策略
重点在合规非价格；防关联串标认定；报价贴近限价留空间
决策清单
  • 邀请资格合法取得？
  • 与其他方有关联关系？
  • 文件制作痕迹独立？
  • 报价无规律性差异？

1.3 竞争性磋商

服务与复杂项目首选
适用场景
  • 政府购买服务
  • 技术复杂/性质特殊
  • PPP / 专利专有技术
核心机制
  • 多轮磋商
  • 最后报价密封提交
  • 综合评分法
死亡陷阱
  • 磋商文件响应不彻底（逐条响应）
  • 最终报价超磋商范围即废
  • 磋商轮次控节奏失误
  • 提交最后报价超时
  • 磋商意见未书面留痕
报价策略
首轮高10-15%留空间；末轮一锤定音；关注磋商组倾向
决策清单
  • 磋商文件逐条对应？
  • 最终报价在磋商范围内？
  • 要求书面留痕？
  • 权重是否清晰？

1.4 竞争性谈判

紧急 / 技术不确定
适用场景
  • 紧急采购（抢险救灾）
  • 技术方案不确定
  • 招标后无合格标
核心机制
  • 多轮谈判
  • 最低价中标
  • 谈判后最后报价
死亡陷阱
  • 报价策略失误差价被砍
  • 谈判泄露底价（被对手参考）
  • 最后报价超采购预算即废
  • 技术方案调整未响应
  • 多家同价再报/抽签
报价策略
首轮高成本30-40%；末轮贴成本底线；不主动透露成本
决策清单
  • 采购预算已知？
  • 谈判轮次上限？
  • 技术方案可调整？
  • 报价在成本底线之上？

1.5 询价

标准现货货物
适用场景
  • 规格标准统一
  • 现货货源充足
  • 金额较小
核心机制
  • 3家以上一次报价
  • 不得更改
  • 最低价中标
死亡陷阱
  • 询价文件响应不全即废（逐项）
  • 报价超采购预算
  • 一次定终身无谈判机会
  • 交货期响应不到位
  • 样品/检测报告缺失
报价策略
一报价即终价精准报；算清成本+利润；报"次低"防质疑
决策清单
  • 询价文件逐项响应？
  • 报价在预算内？
  • 交货期可满足？
  • 样品/检测报告齐全？

1.6 单一来源

无竞争 · 高风险
适用场景
  • 只能唯一供应商
  • 紧急情况不可预见
  • 保证一致性/配套
核心机制
  • 无竞争性采购
  • 直接议价
  • 公示期社会监督
死亡陷阱
  • 价格合理性被质疑（对标行情）
  • 公示期遭异议/举报即终止
  • "唯一性"证据链不完整
  • 审批层级不够（省级财政）
  • 关联交易嫌疑
报价策略
价格对标3份行情；成本构成透明；首轮高20%留空间
决策清单
  • 唯一性证据链完整？
  • 价格合理性材料齐全？
  • 异议应对预案？
  • 审批层级合规？无关联？

第二章五大行业专属坑

2.1 工程招标

施工 / 勘察 / 设计 / 监理 / 工程总承包
围标串标认定
  • 电子标：IP/MAC/CPU序列号/硬盘序列号重合 → 直接认定
  • 文件痕迹：创建人/修改人/保存人一致 → 嫌疑
  • 报价规律性：等差/等比数列 → 嫌疑
  • 文件内容雷同：方案/错误表述一致 → 认定
项目经理锁定
  • 一造一建不能同时挂多个在建项目
  • 社保连续缴纳（通常6个月以上）
  • 证书有效且注册单位与投标人一致
  • 中标后不得更换（更换需建设单位同意）
类似业绩认定
  • 合同 + 竣工验收证明双件齐全
  • 金额/规模/类型匹配"类似工程"定义
  • 时间范围通常近3-5年
  • 部分项目要求业绩项目经理与本次一致
履约保证金
比例 ≤ 中标合同金额10%；方式：银行保函/担保保函/现金；退还：验收合格后28天内
农民工工资保证金
工程类强制；单独账户、专款专用；未存储影响开工
应对策略
  • 投标文件独立制作：独立IP/设备/人员，杜绝痕迹重合
  • 项目经理查档：投标前"四库一平台"核查在建情况
  • 业绩双证齐全：合同+竣工验收缺一不可
  • 履约保证金优先银行保函（占用授信少）

2.2 货物招标

设备 / 材料 / 系统集成 / 软硬件采购
参数控标
  • "★"号条款（实质性）逐条响应，缺一条即废
  • 参数倾向性：特定品牌参数需用"或同等品质"软化
  • 参数负偏离：低于要求即失分，重大负偏离即废
  • 关键技术参数须第三方检测报告支撑
原厂授权/代理
  • 代理商必须提供原厂授权书
  • 授权层级：一级/二级代理权限不同
  • 有效期：覆盖投标有效期+履约期
  • 售后承诺函：原厂或授权机构出具
检测报告时效
通常3年内有效（部分1年）；机构须CMA/CNAS；参数对应、型号一致
知识产权连带
侵权投标人担全责；软件需正版/著作权；硬件需专利避免仿冒；中标后兜底
交货期与调试
交货期为实质性条款；安装调试责任/费用/验收划分；培训要求；备品备件清单
应对策略
  • 参数响应表逐条核对：每★号条款须"满足/优于"
  • 检测报告提前梳理：3年内CMA/CNAS，参数型号一致
  • 原厂授权提前申请：层级/有效期/范围核对
  • 知识产权自查：专利/软著避免侵权

2.3 服务招标

咨询 / 物业 / 安保 / IT服务 / 政府购买服务
人员证书得分
  • 一建/PMP/高级职称/信息系统项目管理师/注册咨询师
  • 证书有效且注册单位与投标人一致
  • 数量/等级/专业方向匹配评分办法
  • 社保连续缴纳（通常6个月-1年）
负责人履历
  • 类似项目经验：规模/类型/角色
  • 工作年限：通常5-10年以上
  • 证书：高级职称+行业执业资格
  • 无在建冲突项目
类似业绩证明
合同 + 用户证明双件齐全；用户证明需甲方盖章含服务内容/期限/质量评价；金额/类型/期限匹配
社保真实性
核心人员社保连续（6个月以上）；缴纳单位须一致；临时挂证易查出；变更需建设单位同意
方案可执行性
忌空话套话；须有流程/人员配置/应急预案；标准量化（响应/解决时间）；承诺可考核（KPI/违约）
应对策略
  • 人员证书矩阵化管理：证书/社保/履历建卡避免复用
  • 业绩档案双件齐全：按行业/规模/金额分类归档
  • 社保一致性核查：投标前查所有核心人员缴纳单位
  • 方案模板化+个性化：避免千篇一律低分

2.4 政府采购

各级国家机关 / 事业单位 / 团体组织 · 财政性资金
中小企业声明函
  • 强制提交（所有政采项目）
  • 价格扣除：小微6-10%（工程5-6%）
  • 联合体：大中型+小微，2-3%扣除
  • 造假 → 进黑名单
进口产品审批
原则采国货；确需进口：财政部审批+专家论证+公示；关境外生产算进口；中外合资不算
节能产品清单
强制清单：须采购清单内产品；优先清单：节能/环保/节水优先；半年更新；投标前核对版本
信用中国查询
不得为失信被执行人/重大税收违法/政采严重违法失信；查询投标截止日前3年内
政策功能
监狱企业/残疾人单位视同小微；支持脱贫农副产品；创新产品首购订购
应对策略
  • 声明函模板化：严格按财政部模板，数据真实
  • 进口审批提前规划：周期2-3个月
  • 节能清单实时更新：投标前核对产品在列
  • 信用中国截图存档：投标前1周内查询

2.5 EPC / PPP

工程总承包 / 政府和社会资本合作 / 特许经营
联合体责任
协议明确设计/施工责任边界；各方承担连带责任；牵头方代表投标/签约/履约；成员变更需同意且资质不低于原
融资可行性
  • 银行授信函：覆盖总投资50%以上
  • 融资方案：来源/结构/还款计划
  • 自有资金：覆盖20-30%
  • 能力评估：资产负债率/流动比率/现金流
绩效与付费
按绩效付费：可用性+运营绩效；建设期/运营期分别设指标；付费风险（能力/延迟）；调价机制（物价/汇率/利率）
全周期成本
建设+运营+维护+拆除；折现率6-8%；周期15-30年；精度±10%以内
风险分担
政策风险政府担；汇率政府担/共担；工期社会资本担；不可抗力共担；须在合同中明确
应对策略
  • 联合体协议精细化：责任/权利/收益明确
  • 融资方案提前对接银行：授信/承诺/意向函分层
  • 绩效指标分别谈判：避免指标过高
  • 成本测算模型化 + 风险逐项谈

第三章决策前哨 · 四步定盘

拿到招标文件，第一件事不是写标书，而是用四步法定盘——判断该不该投。

STEP 01

定方式 · 属于哪种采购

是否使用财政性资金？
├─ 是 → 政府采购法管辖
│   ├─ 限额以上 → 公开招标（默认）
│   ├─ 特殊性质 → 邀请招标
│   ├─ 技术复杂/服务类 → 竞争性磋商
│   ├─ 紧急/技术不确定 → 竞争性谈判
│   ├─ 标准现货货物 → 询价
│   └─ 唯一供应商 → 单一来源
└─ 否 → 招标投标法管辖
    ├─ 限额以上 → 公开招标（默认）
    ├─ 特殊性质 → 邀请招标
    └─ 其他方式参照政府采购
  • 资金性质：财政性 vs 国有 vs 社会
  • 项目金额：是否达限额
  • 项目性质：保密/紧急/技术复杂
  • 供应商数量：是否唯一
STEP 02

定行业 · 属于哪个赛道

采购对象是什么？
├─ 工程建设（施工/勘察/设计/监理）→ 工程招标
├─ 货物（设备/材料/系统）→ 货物招标
├─ 服务（咨询/物业/IT/安保）→ 服务招标
├─ 财政性资金采购 → 政府采购（叠加政策功能）
└─ 工程总承包/特许经营 → EPC / PPP
  • 采购对象：工程/货物/服务/综合
  • 是否叠加政采政策功能
  • 是否涉及融资/运营/绩效付费（EPC/PPP特征）
STEP 03

查废标条款 · 逐条核

废标条款来源：①"投标无效条款"章节 ②"实质性条款"清单（★号） ③"资格要求"章节 ④法律法规强制性规定

  • 资质证书齐备且有效？
  • 业绩达标（合同+验收/用户证明）？
  • 人员证书齐全且可锁定？
  • 实质性条款逐条响应？
  • 投标保证金按时到账？
  • 投标文件按时递交？
  • 中小企业声明函合规（政采）？
  • 信用中国无失信记录？
STEP 04

算匹配度 · 资质/业绩/人员/财力

维度权重评估要点
资质匹配度25%资质等级、专项资质、行业许可
业绩匹配度30%类似项目数量、金额、类型、时间
人员匹配度30%项目经理、技术负责人、核心团队证书
财力匹配度15%营收规模、资产负债率、现金流、融资能力
匹配度阈值： ≥ 80分 高匹配必投 ｜ 60-80分 中匹配评估后决策 ｜ < 60分 低匹配建议放弃（除非有定向优势）
